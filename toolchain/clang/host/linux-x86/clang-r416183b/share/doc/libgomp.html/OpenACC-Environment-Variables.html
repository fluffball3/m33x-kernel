7 OpenACC Environment Variables

The variables ACC_DEVICE_TYPE and ACC_DEVICE_NUM are defined by section 4 of the OpenACC specification in version 2.0. The variable ACC_PROFLIB is defined by section 4 of the OpenACC specification in version 2.6. The variable GCC_ACC_NOTIFY is used for diagnostic purposes.

  • ACC_DEVICE_NUM
  • ACC_PROFLIB
